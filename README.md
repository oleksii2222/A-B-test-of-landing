# A-B-test-of-landing
Проведене а/б тестування початкової сторінки, яку клієнти бачать при переході на сайт. У компанії було використано 3 дизайни даної сторінки, і моя задача була провести а/б тест для виялвення найбільш вдалого дизайну. У ході дослідження було сформовано воронки конверсії користувачів по всі стадіям від лендінгу до покупки.
