# A/B-тестування Landing Page

![Порівняння ефективності Landing Page](https://media.discordapp.net/attachments/758656190413930516/1517520935387795516/image.png?ex=6a369501&is=6a354381&hm=79e5ef09807727d8b869eb6e6e2308adb3c05779c87e22e4fd4ea69e5cb33bf0&=&format=webp&quality=lossless&width=1418&height=810)

## Опис проєкту

У цьому проєкті проведено **A/B-тестування початкових сторінок (Landing Pages)**, які користувачі бачать після переходу на сайт. Компанія використовувала три варіанти дизайну: **landing-1**, **landing-2** та **landing-3**. Метою дослідження було визначення найбільш ефективного дизайну, який забезпечує найвищу конверсію в покупку.

Для аналізу було побудовано **маркетингові воронки продажів**, що відображають шлях користувача від першого відвідування сайту до завершення покупки. Також проведено **статистичну перевірку гіпотез за допомогою двовибіркового Z-тесту для пропорцій**.

## Мета дослідження

Визначити, який із трьох варіантів Landing Page забезпечує найкращу ефективність та максимізує кількість завершених покупок.

## Результати аналізу

| Landing Page | Відвідувачі | Покупки | Конверсія |
|--------------|------------|---------|-----------|
| **landing-1** | 48 тис. | 2 тис. | **4,53%** |
| **landing-2** | 131 тис. | 10 тис. | **7,72%** |
| **landing-3** | 79 тис. | 3 тис. | **3,40%** |

### Основні висновки

- **landing-2** продемонстрував найкращий результат із конверсією **7,72%**.
- **landing-1** забезпечив конверсію на рівні **4,53%**.
- **landing-3** показав найнижчу ефективність — **3,40%**.
- Найбільша частка користувачів успішно проходила всі етапи воронки саме через **landing-2**.

## Статистичне тестування

Для перевірки статистичної значущості різниці між **landing-1** та **landing-2** було проведено **двовибірковий Z-тест для пропорцій**.

**Результати тесту:**

- Конверсія **landing-1**: **4,53%**
- Конверсія **landing-2**: **7,72%**
- **Z-statistic = -23.54**
- **p-value < 0.001**

Оскільки **p-value < 0.05**, нульову гіпотезу було відхилено. Різниця між конверсіями є **статистично значущою**, а **landing-2** демонструє суттєво кращу ефективність.

## Висновок

Проведений аналіз показав, що **landing-2** є найбільш ефективним варіантом стартової сторінки. Він має найвищу конверсію в покупку, забезпечує краще проходження користувачів через усі етапи воронки та статистично значуще перевершує інші варіанти.

**Рекомендація:** використовувати **landing-2** як основний дизайн Landing Page для підвищення конверсії та збільшення продажів.
